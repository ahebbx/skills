# 菜品库

按场景分类。标签：〔素〕〔荤〕〔清淡〕〔重口〕〔汤〕。不必局限于此库，这只是灵感起点。

## 自己做 · 快手菜（≤20分钟，累了/工作日首选）
- 番茄炒蛋〔素·清淡〕/ 番茄豆腐汤〔素·汤〕
- 蒜蓉油麦菜、清炒西兰花、手撕包菜〔素〕
- 青椒肉丝、京酱肉丝、蒜苔炒肉〔荤〕
- 香煎鸡胸/三文鱼 + 焯西兰花〔荤·清淡〕健身友好
- 韭菜炒蛋、秋葵炒蛋〔素〕
- 麻婆豆腐〔重口〕、干煸四季豆〔重口〕
- 虾仁滑蛋、白灼虾〔荤·清淡〕
- 丝瓜蛋汤、紫菜蛋花汤、冬瓜虾皮汤〔汤·清淡〕

## 自己做 · 一锅出主食（懒人友好，不用配菜）
- 番茄牛腩面 / 葱油拌面 / 炸酱面 / 酸汤肥牛面
- 蛋炒饭 / 腊肠煲仔饭 / 咖喱鸡肉饭 / 亲子丼
- 疙瘩汤、皮蛋瘦肉粥
- 饺子/馄饨（速冻也算，配个凉拌菜就很好）

## 自己做 · 硬菜（周末/犒劳/来客人）
- 红烧肉、可乐鸡翅、黄焖鸡、大盘鸡
- 糖醋排骨、粉蒸排骨、萝卜炖牛腩
- 酸菜鱼、清蒸鲈鱼、红烧带鱼
- 羊蝎子、猪肚鸡、腌笃鲜〔汤〕
- 辣子鸡、水煮牛肉〔重口〕

## 外卖品类（按状态选）
- 想省事吃饱：黄焖鸡米饭、猪脚饭、烧腊饭、盖浇饭、麻辣香锅
- 想清淡：轻食沙拉、粥品、蒸菜馆、云南米线（清汤）
- 想重口过瘾：麻辣烫、川湘菜、烤肉饭、炸鸡、螺蛳粉
- 想换口味：日料便当、泰式咖喱、意面、越南粉、墨西哥卷
- 深夜/胃不舒服：粥、汤面、馄饨

## 外食品类（社交/犒劳/懒得洗碗）
- 二人/多人：火锅、烤肉、烤鱼、串串、湘菜馆、粤式茶餐厅
- 一人食：日式拉面、寿司、牛肉面、饺子馆、快餐简餐
- 犒劳自己：日料放题、牛排、粤菜海鲜、omakase、网红新店
- 快速解决：食堂、商场地下美食广场、便利店关东煮

## 早餐
- 楼下买：包子豆浆、煎饼果子、鸡蛋灌饼、肠粉、豆腐脑、油条
- 在家快手：燕麦牛奶+香蕉、三明治+煎蛋、吐司+牛油果、酸奶+坚果、水煮蛋+玉米
- 有闲情：小米粥+小菜、鸡蛋饼、法式吐司

## 营养缺口 → 补什么
| 最近记录的信号 | 推荐方向 |
|---|---|
| 连续外卖/重油重盐 | 在家清淡小炒 + 汤 |
| 蔬菜少 | 大份绿叶菜（蒜蓉炒/白灼/凉拌）、蔬菜汤 |
| 蛋白质少 | 鱼虾、鸡胸、豆腐、蛋 |
| 红肉偏多 | 换鱼虾/禽类/豆制品 |
| 碳水轰炸（面、饭、饼连吃） | 菜+蛋白为主，主食减半 |
| 最近没喝过汤 | 加个汤（10分钟蛋花汤也算） |
